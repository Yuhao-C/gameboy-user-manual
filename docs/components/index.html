Components

Learn About the Componentsâ€‹

  1. Infrared Communication Port (COMM) - Communicate with another Game Boy Color. (Available in some games only. See Playing Two-Player Games with the COMM Port.)

  2. Power Indicator (POWER) - Lights when the device is on. This will begin to dim as the battery runs low.

  3. Color LCD Screen - View your game play on a high quality color screen.

  4. Operation Buttons - Control your game play. See game instructions for button functions.

  5. External Power Supply Jack - Connect to Game Boy pocket AC Adapter to play without batteries.

  6. Game Pak Slot - Insert a Game Pak into the slot to play the game. See Playing Games on the Game Boy Color.

  7. Wrist Strap Attachment - Attach a wrist strap to firmly hold your Game Boy Color.

  8. Power Switch (ON/OFF) - Slide the switch towards the top to turn on power. Always insert a Game Pak before turning on the power.

  9. External Extension Connector (EXT) - Connection to another Game Boy with a Game Link Cable. See Playing Two-Player Games with the Game Link Cable.

  10. Volume Dial (VOL) - Rotate down to increase the volume. Rotate up to decrease the volume.

  11. Battery Cover - Remove cover to add or replace batteries.

  12. Speaker - Plays sound when headphones are not plugged in.

  13. Headphone Jack - Connect to stereo headphones for better audio experience.
