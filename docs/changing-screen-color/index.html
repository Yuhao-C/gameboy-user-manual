Changing Screen Color

info

Changing the color palette feature will only work with the original Game Boy (monochrome) Game Paks. The color cannot be changed after the game title screen appears, or on Game Boy Color Type 2 or 3 Game Paks as described in Compatible Game Paks.

There are 12 different color palettes that can be selected when using the original Game Boy Paks.

  1. To change the color palette, first insert a Game Pak and turn on the power as described in Using the Video Game System.

  2. When the Game Boy logo appears, press the + button and button combination shown below.

    PRESS
    COLOR
    Brown
    Red
    Dark Brown
    Pale Yellow
    Orange
    Yellow
    Blue
    Dark Blue
    Gray
    Green
    Dark Green
    Reverse
